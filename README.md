Gewefope [![Build Status](https://travis-ci.org/Gewefope/gewefope.svg?branch=master)](https://travis-ci.org/Gewefope/gewefope)
========

Website with weather forecast.

Build and run
```
npm install -g gulp
npm install
gulp
node server.js
```

Run tests
```
npm test
```
